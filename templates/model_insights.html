Data Sources — Real vs Synthetic
✅ Real data: Zimbabwe national maize yield history from FAOSTAT (2000–2024). Seasonal weather (rainfall, temperature, humidity) from NASA POWER satellite reanalysis per province coordinates.
⚠️ Synthetic augmentation: ENSO index (from real historical NOAA classifications), soil moisture estimate (derived from rainfall), NDVI proxy (derived from rainfall), fertilizer rate, and planting date were generated using agronomically-calibrated distributions. This is standard practice for prototype ML models where comprehensive farm-level data does not yet exist, and is explicitly disclosed here. Synthetic features will be replaced with real farm observations as RimAI scales.

Honest Limitations
  • National-level ground truth: FAOSTAT provides national averages. Province-level predictions are derived using agronomic zone priors, not measured provincial data.
  • Synthetic augmentation: R² of {{ meta.cv_r2 }} and {{ (meta.risk_classifier_accuracy * 100) | round(1) }}% accuracy reflect genuine cross-validated performance on augmented data. These metrics will be recalibrated as real farm-level data is collected.
  • Class imbalance: The training set has more Low-risk observations (149) than High (34), which suppresses High-risk precision. Future versions will apply SMOTE or weighted classes.
  • Improvement pathway: Each season of real farm data collected through RimAI replaces synthetic features. The architecture is built for this — synthetic augmentation is a starting point, not a permanent design.
